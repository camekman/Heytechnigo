A term that originally referred to open source software (OSS).

Open Source is code that is designed to be publicly accessible. Anyone can see, modify, and distribute the code as they see fit.

Developed in a collaborative way it is often more flexible, cheaper and has more longevity than its peers.

An open source development model is the process used by an open source community project to develop open source software. The software is then released under an open source license, so anyone can view or modify the source code. Â
